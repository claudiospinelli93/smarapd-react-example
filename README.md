## Instalação
Clone o repositório e rode os comandos na pasta raiz do projeto:

    npm i yarn -g
    cd Common
    yarn install
    cd ../WebApp
    yarn install

## Execução

Para executar o projeto, utilize o **comando** na pasta **WebApp**:
(Será criado um webapp na url: http://localhost:8081)

    yarn start
